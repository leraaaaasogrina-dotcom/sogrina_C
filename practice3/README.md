тут будут массивы   
1. Вывод массива в обратном порядке
Пользователь вводит с клавиатуры 10 целых чисел.
Нужно сохранить их в массив, а затем вывести на экран в обратном порядке: сначала последнее введённое число, затем предпоследнее и так далее, до первого. (1.c)

2. Поиск максимального элемента массива
Сначала пользователь вводит целое число n — количество элементов массива (например, от 1 до 100).
Затем он вводит n целых чисел.
Нужно:
найти наибольшее (максимальное) из этих чисел;
вывести на экран только это максимальное значение.

3. Минимальный элемент и его индекс
Пользователь вводит n — количество элементов массива, затем n целых чисел.
Нужно:
найти наименьшее (минимальное) значение в массиве;
найти индекс этого элемента (номер в массиве, начиная с 0);
вывести на экран минимальное значение и его индекс.
Если минимум встречается несколько раз, можно вывести индекс первого из них.

4. Сумма всех элементов массива
Пользователь вводит n — количество элементов массива, затем n целых чисел.
Нужно:
посчитать сумму всех введённых чисел;
вывести полученную сумму на экран как одно число.

5. Количество отрицательных элементов
Пользователь вводит n — количество элементов массива, затем n целых чисел (могут быть положительные, ноль или отрицательные).
Нужно:
посчитать, сколько среди элементов массива чисел меньше нуля;
вывести это количество на экран.

6. Замена отрицательных элементов на ноль
Пользователь вводит n — количество элементов массива, затем n целых чисел.
Нужно:
заменить в массиве все отрицательные числа на 0;
вывести получившийся массив на экран (все элементы через пробел).
Порядок остальных (неотрицательных) элементов должен сохраниться.

7. Среднее арифметическое элементов массива
Пользователь вводит n — количество элементов массива, затем n целых чисел.
Нужно:
вычислить среднее арифметическое: сумму всех элементов разделить на n;
вывести среднее арифметическое как число с плавающей точкой (например, double).

8. Проверка наличия хотя бы одного чётного числа
Пользователь вводит n — количество элементов массива, затем n целых чисел.
Нужно:
проверить, есть ли в массиве хотя бы одно чётное число (делится на 2 без остатка);
если есть хотя бы одно чётное, вывести на экран, например, YES;
если чётных чисел нет, вывести NO.

9. Проверка, упорядочен ли массив по возрастанию
Пользователь вводит n — количество элементов массива, затем n целых чисел.
Нужно:
проверить, идут ли элементы массива по возрастанию или неубыванию, то есть выполняется ли для всех соседних пар условие: предыдущий элемент не больше следующего (a[i] ≤ a[i+1]);
если массив упорядочен по возрастанию (неубывающе), вывести YES;
иначе вывести NO.

10. Обмен первого и последнего элементов массива
Пользователь вводит n — количество элементов массива, затем n целых чисел.
Нужно:
поменять местами первый и последний элементы массива;
вывести изменённый массив на экран.
Пример: было 1 2 3 4 5 → станет 5 2 3 4 1.

11. Подсчёт количества вхождений заданного числа
Пользователь вводит n — количество элементов массива, затем n целых чисел.
После этого пользователь вводит ещё одно число x.
Нужно:
посчитать, сколько раз число x встречается среди элементов массива;
вывести это количество на экран.
Если число не встречается ни разу, нужно вывести 0.

12. Поиск индекса первого отрицательного числа
Пользователь вводит n — количество элементов массива, затем n целых чисел.
Нужно:
найти первый (самый левый) элемент массива, который меньше нуля;
вывести его индекс (номер в массиве, начиная с 0).
Если в массиве нет ни одного отрицательного числа, вывести, например, −1.

13. Циклический сдвиг массива вправо на одну позицию
Пользователь вводит n — количество элементов массива, затем n целых чисел.
Нужно выполнить циклический сдвиг элементов вправо на одну позицию:
последний элемент массива должен стать первым;
каждый остальной элемент сдвигается на одну позицию вправо.
Пример: 1 2 3 4 5 → 5 1 2 3 4.
