1. Что такое файл в языке C?

Файл — это последовательность байтов, хранящаяся на диске (или другом носителе). В C файл представляет собой либо текстовые данные, либо бинарные данные, с которыми программа может работать через специальные функции.
2. Что такое FILE*?

FILE* — это указатель на структуру FILE, которая содержит информацию о файле: его текущую позицию, режим открытия, буфер и т.д. Через этот указатель программа обращается к файлу. FILE *file = fopen("data.txt", "r");
3. Какие режимы открытия файла существуют?

Режим	Описание
"r"	    чтение (файл должен существовать)
"w"	    запись (создаёт новый или перезаписывает)
"a"	    добавление в конец файла
"r+"	чтение и запись (файл должен существовать)
"w+"	чтение и запись (создаёт новый или перезаписывает)
"a+"	чтение и добавление в конец
"rb"	чтение бинарного файла
"wb"	запись бинарного файла
"ab"	добавление в бинарный файл

4. Чем отличается текстовый файл от бинарного?

Текстовый файл	                            Бинарный файл
Содержит читаемые символы	        Содержит байты в том виде, как в памяти
Можно открыть в блокноте	        Нечитаем для человека
Символы \n могут преобразовываться	    Никаких преобразований
Медленнее для чтения/записи	        Быстрее
Используется для конфигов, логов	Используется для структур, массивов

6. Что делают fgets и fputs?

Функция	                   Что делает	                                Пример
fgets(буфер, размер, файл)	читает одну строку из файла	        fgets(str, 100, file);

7. Что делают fprintf и fscanf?

Функция	                                Что делает	                    Аналог
fprintf(файл, "формат", ...)	форматированная запись в файл	        printf
fscanf(файл, "формат", ...)	    форматированное чтение из файла	        scanf
fprintf(file, "Имя: %s, Возраст: %d\n", name, age);
fscanf(file, "Имя: %s, Возраст: %d", name, &age);

8. Что делают fread и fwrite?

Функция	                                            Что делает	Параметры
fwrite(данные, размер, количество, файл)	запись бинарных данных	    fwrite(&p, sizeof(Person), 1, file);
fread(буфер, размер, количество, файл)	    чтение бинарных данных	     fread(&p, sizeof(Person), 1, file);
Используются для работы со структурами и массивами напрямую (без преобразования в текст).

9. Что делают fseek и ftell?

Функция	                                    Что делает	                Пример
fseek(файл, смещение, откуда)	перемещает указатель в файле	fseek(file, 10, SEEK_SET);
ftell(файл)	                   возвращает текущую позицию       long pos = ftell(file);
Параметры fseek:

SEEK_SET — от начала файла
SEEK_CUR — от текущей позиции
SEEK_END — от конца файла
