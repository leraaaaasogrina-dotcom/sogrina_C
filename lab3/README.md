                    Лабораторная работа 3
                Динамические массивы в языке C
Цель работы: изучить работу с динамической памятью: функции malloc, calloc, realloc, free, создание и использование динамических массивов.

1. Что делает malloc- Выделяет блок памяти указанного размера.  Память не обнуляется. 
2. Чем отличается malloc от calloc?
    malloc - выделяет память, не обнуляет, 1 аргумент.
    calloc-выделяет память, обнуляет (все байты становяться 0), 2 аргумента.
3. Что делает realloc?
Изменяет размер уже выделенной памяти:
 может увеличить
 может уменьшить
 старые данные сохраняются
4. Что произойдёт при утечке памяти?
Программа занимает память, но не освобождает её.
Со временем памяти может не хватить - программа упадёт или компьютер зависнет.
5. Зачем нужен free?
Освобождает выделенную память, чтобы её можно было использовать снова.
Без free - утечка памяти.
6. Что произойдёт при двойном освобождении памяти?
Ошибка-(программа может упасть или повредить данные).
Нельзя вызывать free дважды для одного указателя.
7. Как проверить, что malloc вернул NULL?

Сравнить указатель с NULL: 
    int n;
    int *arr = (int*)malloc(n * sizeof(int));
    if(arr == NULL) {
    printf("Ошибка выделения памяти!");
    }
8. В чём отличие статического и динамического массива?
Статический массив-
    Размер известен до компиляции
    int arr[10];
    Освобождается автоматически
    Размер нельзя изменить
Динамический массив-
    Размер задаётся во время работы
    int *arr = malloc(n * sizeof(int));
    Нужно вызывать free()
    Можно изменить через realloc
